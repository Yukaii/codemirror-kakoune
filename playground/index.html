Try these keys

  • i enter insert mode
  • Esc return to normal mode
  • h j k l move the cursor
  • w b e move by word
  • x select the current line
  • d delete the current selection
  • y yank, p paste
  • f t F T find a character on the line
  • % select the whole buffer, , clear extra selections

The key processor is intentionally small right now. It is structured so the Kakoune command set can grow without changing the playground shell.
